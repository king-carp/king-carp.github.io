一、计算机硬件组成与工作原理

首先Python是一门编程语言

​ 那么我们需要了解什么是编程语言。其实编程语言就如同我们学习的汉语，英语，法语等等，都是沟通的语言。编程语言是与计算机沟通的语言。

​ 比如我们需要做这样的一件事情：某个客户去银行取钱，我们作为银行管理员要对新来的员工讲解取钱流程。

​ 银行———————汉语———————–员工

​ 让用户输入账号

​ 让用户输入密码

​ 帮用户取钱

​ 如果用户在ATM机上取钱时，我们的流程与逻辑应该是一样的。

​ 程序员——————编程语言——————ATM机

​ 让用户输入账号

​ 让用户输入密码

​ 帮用户取钱

人类为何要与计算机沟通

​ 我们的目的是让计算机帮助人类工作。因为计算机不需要睡眠等原因工作效率比人要高。

什么是编程***

​ 编程就是把做事的顺序/业务逻辑用编程语言表达出来

什么是程序***

​ 我们存放程序代码的文件就是程序，也可以叫软件

计算机三大核心硬件***

​ 原则：所有的计算机硬件都是在模仿人的某一个功能或者器官

​ 1.CPU：相当于人的大脑，主要负责逻辑运算与数学运算，执行指令。控制器+运算器=CPU

​ 2.内存：相当于人类大脑的记忆功能，用于临时存储数据，断电数据就丢失了

​ 3.硬盘：相当于我们用笔写下来的笔记，可永久存储数据，断电数据不会丢失

​ 内存： 优点：传输速度快 缺点：只能临时存储数据

​ 硬盘： 优点：永久存储数据 缺点：传输速度慢

程序运行与计算机三大核心硬件的关系***

​ 程序首先的存储在硬盘上的，当程序运行时，先将程序代码从硬盘读入内存，然后CPU从内存读取指令开始运行

I/O问题与程序优化思路

​ 1.读

​ 能从内存读取就不从硬盘读

​ 当某些数据需要经常使用，应该提前从硬盘加载到内存中

​ 2.写

​ 尽量攒一大波数据一次性写入内存中，减少写入次数

​ Input：输入设备，例如键盘，鼠标

​ Output：输出设备，如显示屏，打印机

计算机单位*

​ 8bit = 1Byte

​ 1024Byte = 1Kb

​ 1024Kb = 1Mb

​ 1024Mb = 1Gb

​ 1024Gb = 1Tb

X86-64代表什么***

​ x86代表CPU的型号，目前日常使用的计算机都是x86型号的CPU。64位代表CPU一次性读取64bit的数据运行

CPU的两种工作状态***

​ 1.内核态：运行操作系统程序，控制计算机的硬件，cpu可以访问内存的所有数据，包括外围设备，例如硬盘，网卡。

​ 2.用户态：运行用户程序，只能受限的访问内存，且不允许访问外围设备，占用cpu的能力被剥夺，cpu资源可以被其他程序获取
