Python day2:计算机硬件基础2

一、计算机硬件基础知识普及

存储器*

RAM：内存中可读可写的部分

ROM：可充当内存，存放BIOS(Basic Input Output System)

CMOS：可充当硬盘，断电数据丢失，但是耗电量极低

CPU，ROM，CMOS用来运行BIOS程序

硬盘详解

磁道，扇区，柱面三部分。

我们知道，机械硬盘的存取IO速度慢比固态硬盘慢。其实无论是机械硬盘还是固态硬盘，如果已经开始传输数据，那么速度相差不会太大。

假设某一个硬盘转速为7200转/分钟，那么我们可以算出转一圈的速度为8ms。同时我们需要知道对于机械硬盘，磁头不可能每次都能找到磁道的起始点，我们有平均延迟时间与平均寻道时间。那么我们进行一个简单的计算，根据8ms转一圈，平均寻道时间为4ms，平均寻道时间大概为5ms。那么等待的时间为9ms。机械硬盘的慢主要是在这部分的等待时间。

二、操作系统***

什么是操作系统***

操作系统是一个协调，管理，控制计算机硬件资源与应用软件资源的控制程序。

平台***

平台 = 计算机硬件 + 操作系统。一款优秀的软件，跨平台性很重要。

计算机体系的三层结构***

上层：应用程序，同时用户也可直接使用操作系统

中层：操作系统（Windows，Mac，Linux）

底层：计算机硬件（CPU，内存，硬盘）

三、编程语言分类**

机器语言

机器语言：直接用二进制指令编写程序

优点：运行效率高

缺点： 1.二进制指令太难记忆

​ 2.一个简单的功能都需要很多步骤才能完成，开发效率低

汇编语言

汇编语言：用英文标签取代二进制指令编写

优点：1.运行效率依然高，但是低于机器语言

​ 2.解决了机器语言难记的问题

缺点：只是解决了二进制难记的问题，一个简单功能依然需要很多步骤，开发效率依然较低。

高级语言

高级语言：用人类能理解的表达方式去编写程序，计算机无法直接理解，需要翻译工具。

编译型

编译型：利用编译器把源程序的每一条语句都编译成机器语言，并保存成二进制文件，这样计算机运行该程序时可以直接以机器语言来运行此程序，不再需要编译器，运行速度很快

翻译工具：编译器

代表语言：C语言，Go语言

解释型

解释型：每次在执行程序时，计算机利用解释器一条一条的将代码解释成机器语言给计算机来执行

翻译工具：解释器

代表语言：Python

各种语言比较

执行效率：机器语言>汇编语言>高级语言（编译型>解释型）

开发效率：机器语言<汇编语言<高级语言（在高级语言中，开发效率取决于语言本身）
